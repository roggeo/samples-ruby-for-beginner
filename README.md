# Samples Ruby for beginner

## Install

1. Get [download](https://github.com/roggeo/samples-ruby-for-beginner/archive/master.zip)

2. Unzip the samples-ruby-for-beginner_master.zip

## Running

	cd samples-ruby-for-beginner_master
	ruby [some_file].rb
